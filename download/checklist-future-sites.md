# Чек-лист: Создание идеального сайта на Next.js

> Для каждого нового проекта проверяем все пункты ниже.

---

## 1. СКОРОСТЬ И ПРОИЗВОДИТЕЛЬНОСТЬ ("Чтобы сайт летал")

### Изображения
- [ ] Все изображения через `<Image>` от Next.js (автоматическая оптимизация)
- [ ] Формат WebP/AVIF для фотографий
- [ ] lazy loading для изображений ниже первого экрана
- [ ] Размер постеров не более 200-500 KB
- [ ] Пакет `sharp` установлен (для оптимизации)

### Код и бандл
- [ ] Tree-shaking включён (Next.js по умолчанию)
- [ ] Динамический импорт для тяжёлых компонентов (`next/dynamic`)
- [ ] Минимум сторонних библиотек (только нужные)
- [ ] Font optimization через `next/font` (не внешние CDN)

### Кэширование
- [ ] ISR (Incremental Static Regeneration) для страниц
- [ ] Настроен `revalidate` в `next.config.ts`
- [ ] Статические страницы генерируются при билде

### Сервер
- [ ] Vercel (бесплатный план для старта) или аналог
- [ ] CDN для статики (автоматически на Vercel)
- [ ] HTTP/2 и HTTP/3 (автоматически на Vercel)

### Метрики скорости
- [ ] LCP < 2.5 сек (быстрая загрузка основного контента)
- [ ] FID < 100 мс (быстрый отклик)
- [ ] CLS < 0.1 (нет скачков макета)
- [ ] Проверка через PageSpeed Insights: 90+ на мобильных

---

## 2. SEO (Чтобы быть наверху в поиске)

### Техническое SEO
- [ ] Уникальный `<title>` на каждой странице (40-60 символов)
- [ ] `<meta description>` на каждой странице (100-160 символов)
- [ ] Open Graph теги (og:title, og:description, og:image, og:type)
- [ ] Twitter Card теги
- [ ] Canonical URL на каждой странице
- [ ] `robots.txt` настроен
- [ ] `sitemap.xml` генерируется автоматически
- [ ] JSON-LD структурированные данные (LocalBusiness, Article, FAQ)

### Контентное SEO
- [ ] Главная страница: от 400+ слов уникального текста
- [ ] Каждая страница: минимум 300+ слов
- [ ] H1 — только один на страницу
- [ ] Иерархия заголовков: H1 → H2 → H3 (без пропусков)
- [ ] Альтернативные тексты для всех изображений
- [ ] Внутренняя перелинковка между страницами
- [ ] Блог со статьями (минимум 5-10 статей для старта)

### Ссылочное SEO
- [ ] Регистрация в Яндекс.Вебмастер
- [ ] Регистрация в Google Search Console
- [ ] Карточка в Яндекс.Бизнес
- [ ] Профиль в 2GIS
- [ ] Ссылки с тематических ресурсов

### Локальное SEO (для бизнеса с адресом)
- [ ] Название города/региона в title и description
- [ ] Адрес на каждой странице (футер)
- [ ] Телефон на каждой странице
- [ ] Номера телефонов кликабельны (tel:)
- [ ] Карта на странице контактов (Яндекс.Карты)

## 3. АНАЛИТИКА И МЕТРИКИ

### Обязательные системы
- [ ] Google Tag Manager (GTM) — центральный контейнер
- [ ] Яндекс Метрика (с Вебвизором и Картой кликов)

### Опциональные (по запросу)
- [ ] Facebook Pixel (если есть реклама в FB/Instagram)
- [ ] Google Analytics 4 (через GTM)

### Cookie-баннер
- [ ] Cookie Consent баннер (Принять/Отклонить)
- [ ] Ссылка на политику конфиденциальности

### Переменные окружения (.env)
```
NEXT_PUBLIC_GTM_ID=
NEXT_PUBLIC_YANDEX_METRIKA_ID=
NEXT_PUBLIC_VK_RETARGETING_ID=
```

### Компоненты (включать в каждый проект)
- [ ] `Analytics.tsx` — загрузка скриптов GTM, Метрики, VK
- [ ] `CookieConsent.tsx` — баннер согласия на cookie

## 4. ХОСТИНГ

### Бесплатные / Дешёвые варианты

| Провайдер | Цена | Подходит для | Особенности |
|-----------|------|--------------|-------------|
| **Vercel** | Бесплатно (Hobby) | Next.js | Автодеплой, CDN, SSL, самый быстрый |
| **Netlify** | Бесплатно (Starter) | Статика/SSG | CDN, SSL, формы |
| **GitHub Pages** | Бесплатно | Статика | Только статические сайты |
| **Timeweb** | От 149 руб/мес | Любой хостинг | Российский, поддержка 24/7 |
| **Beget** | От 159 руб/мес | Любой хостинг | Российский, автоустановка |
| **Reg.ru** | От 199 руб/мес | VPS/VDS | Российский, панель управления |

### Рекомендация для Next.js
**Vercel (бесплатный план)** — лучший выбор:
- Автоматический деплой из GitHub
- Мировой CDN (быстрая загрузка везде)
- Бесплатный SSL
- Бесплатный домен `*.vercel.app` для тестов
- Лимиты: 100 GB трафика/мес, unlimited запросов

### Для российских проектов (физ.лица)
**Timeweb Cloud** — от 149 руб/мес:
- Серверы в России (быстро для RU-аудитории)
- Панель управления
- SSL-сертификат бесплатно
- Поддержка Next.js через Node.js

## 5. ДОМЕНЫ

### Недорогие зарубежные домены

| Домен | Регистратор | Цена в год | Особенности |
|-------|-------------|-----------|-------------|
| **.com** | Namecheap, Reg.ru | $8-12 (~700-1000 руб) | Универсальный, самый престижный |
| **.ru** | Reg.ru, Ru-Center | 200-300 руб | Для РФ-аудитории, лучше для Яндекса |
| **.io** | Namecheap | $30-40 (~2500-3500 руб) | Для IT/технологий |
| **.info** | Namecheap | $4-5 (~350-450 руб) | Дешёвый, для информационных сайтов |
| **.site** | Namecheap | $3-4 (~250-350 руб) | Самый дешёвый, для любых проектов |
| **.online** | Namecheap | $2-3 (~180-250 руб) | Ультрабюджетный |
| **.pro** | Reg.ru | $15-20 (~1300-1700 руб) | Для профессиональных услуг |

### Рекомендации по выбору домена
- Для российского бизнеса: **.ru** (лучше ранжируется в Яндексе)
- Для международного проекта: **.com**
- Для бюджетного старта: **.site** или **.online**
- Длина: 8-12 символов, без дефисов, легко диктовать
- Проверить: нет торговых марок, легко произнести по телефону

## 6. БЕЗОПАСНОСТЬ

- [ ] HTTPS (SSL-сертификат) — обязательно
- [ ] Security headers в `vercel.json` (X-Frame-Options, CSP, X-Content-Type)
- [ ] Rate limiting для API-маршрутов
- [ ] Защита от CSRF и XSS (Next.js по умолчанию)
- [ ] Скрытые `.env` файлы (в .gitignore)
- [ ] Нет секретов в клиентском коде

## 7. UX И КОНВЕРСИЯ

### Мобильная адаптация
- [ ] Mobile-first дизайн
- [ ] Все элементы кликабельны на тач-экранах
- [ ] Текст читаем без зума (минимум 16px)
- [ ] Нет горизонтальной прокрутки
- [ ] Проверка на iOS Safari и Android Chrome

### Контакты и заявки
- [ ] Телефон виден на каждом экране (кликабельный)
- [ ] Форма обратной связи (минимум: имя, телефон, сообщение)
- [ ] Кнопка WhatsApp / Telegram в нижнем углу
- [ ] Карта проезда на странице контактов
- [ ] Часы работы и адрес

### Навигация
- [ ] Понятное меню (не более 6-7 пунктов)
- [ ] Хлебные крошки (breadcrumbs)
- [ ] Логотип ведёт на главную
- [ ] 404 страница с навигацией
